goloom's interface supports **internationalization (i18n)**, so the UI can be
presented in different languages.

## Choosing a language

The UI selects a language based on your preferences. Where a translation is
available, screens, labels and messages are localized.

## Contributing translations

Translations live with the frontend. If a language is missing or incomplete,
contributions are welcome through the
[repository](https://github.com/Goloom-App/goloom).
