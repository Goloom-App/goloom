# UI & View Catalog

Detailed breakdown of the primary views in the application and the information they display.

## 1. Dashboard (Overview)
*   **Purpose:** High-level summary of recent activity and performance.
*   **Displayed Information:**
    *   **Upcoming Posts:** A list of the next 5-10 scheduled posts with timestamps and platform icons.
    *   **Recent Performance:** Mini-charts showing engagement trends over the last 7 days.
    *   **Account Status:** A grid of connected accounts with "Active" or "Needs Re-auth" indicators.

## 2. Content Calendar
*   **Purpose:** Visual planning and management of the posting schedule.
*   **Displayed Information:**
    *   **Monthly/Weekly Grid:** Drag-and-drop tiles representing posts.
    *   **Post Preview:** Thumbnails for media-rich posts and text snippets.
    *   **Filters:** Sidebar to toggle visibility by Platform (e.g., Mastodon only) or Account.
    *   **Time Slots:** Highlighted regions showing the "Optimal Posting Times" or "Queue Slots."

## 3. Post Composer (Editor)
*   **Purpose:** The central tool for creating content.
*   **Displayed Information:**
    *   **Unified Text Area:** Main input for the post body.
    *   **Platform Tabs:** Specific sections to override content for Mastodon vs. X (e.g., different character counts).
    *   **Media Gallery:** Grid of attached images/videos with "Alt Text" editing fields.
    *   **Post Preview:** Real-time simulation of the post as it will appear on the mobile/desktop feed of the target platform.
    *   **Scheduling Sidebar:** Date/Time picker and "Add to Queue" toggle.

## 4. Media Library
*   **Purpose:** Management of visual assets.
*   **Displayed Information:**
    *   **Asset Grid:** Thumbnails of all uploaded files.
    *   **Metadata Sidebar:** File size, dimensions, upload date, and "Used in X posts" counter.

## 5. Analytics Dashboard
*   **Purpose:** Deep dive into performance data.
*   **Displayed Information:**
    *   **Metric Selectors:** Toggles for "Total Reach", "Engagement Rate", "Follower Growth".
    *   **Detailed Charts:** Time-series line charts and bar graphs for platform comparisons.
    *   **Top Performing Posts:** A list of posts sorted by engagement (Likes/Shares/Comments).
