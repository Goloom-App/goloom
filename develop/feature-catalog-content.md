# Feature Catalog: Content Creation & Optimization

The content engine is the heart of the application, enabling users to craft, optimize, and organize their social media output.

## 1. Multi-Platform Composer
*   **Unified Drafting:** Create a single post that can be targeted at multiple social media accounts simultaneously.
*   **Platform-Specific Versioning:** Allow users to switch between platform tabs (e.g., "Twitter", "Instagram", "Facebook") to customize the content for each platform's unique constraints (character limits, hashtag rules, image ratios).
*   **Real-time Preview:** Visual representation of how the post will look on the actual social media feed before it is published.

## 2. Media Management
*   **Centralized Media Library:** A workspace-specific repository for images, GIFs, and videos.
*   **Media Editing:** Basic cropping and resizing tools to ensure media meets platform specifications.

## 3. Optimization Tools
*   **Hashtag Manager:** Create and save groups of hashtags (e.g., "Tech", "Photography", "Marketing") for quick insertion into posts.
*   **Templates & Snippets:** Save frequently used text blocks or full post structures as templates.
*   **Dynamic Variables:** Support for placeholders (e.g., `{brand_name}`, `{website_url}`) that are automatically replaced during publishing.
